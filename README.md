# Proyecto Final - Gestión de Tareas con React

Aplicación web desarrollada en React + TypeScript para la gestión de tareas, consumiendo una API REST externa.

## Funcionalidades

- Visualización dinámica de tareas
- Creación de nuevas tareas
- Edición de tareas existentes
- Eliminación de tareas
- Cambio de estado entre Pendiente y Finalizada

## Tecnologías Utilizadas

- React
- TypeScript
- Vite
- Bootstrap
- Axios
- React Router DOM

## API Utilizada

Backend:
https://taskdone-node.onrender.com
